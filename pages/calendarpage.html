This website was developed by Akash Bagchi and Akshaya Nadathur of Dayananda Sagar University - Kudlu Gate, Bangalore as part of the 4th Semester Web Technologies' Mini Project. The main goal of the website was to create a student portal for CSE students wherein there would be easy access to college material, calendars, announcements etc. thereby removing ambiguity and streamlining the lives of thousands of students. This website uses HTML, CSS, JavaScript and JQuery, Bootstrap4, PHP and MySQL to achieve its functionality. We hope the site served your needs!
